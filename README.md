#DAWN-YOLO: Small Object Detection in Low-Light Aerial Images using Degradation Module and P2-Head YOLOv8s
Lightweight YOLOv8-based model with synthetic dark data augmentation for improved detection in night-time drone imagery.

#Contributions
- Lightweight model (only 20.7 MB)
- Improved low-light detection using synthetic data from Degradation Module
- Added P2 detection head for small object detection
- Competitive performance with much lower computation
